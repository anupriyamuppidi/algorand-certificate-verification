# Algorand Certificate Verification System

## 📌 Overview

This project is a decentralized application built on Algorand blockchain to verify academic certificates securely.

## 🎯 Objectives

* Prevent fake certificates
* Provide instant verification
* Maintain immutable records

## ⚙️ Features

* Certificate issuance
* Blockchain storage
* Verification system

## 🛠️ Tech Stack

* Algorand
* Python
* Smart Contracts
